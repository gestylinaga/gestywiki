by: [Gesty Linaga](https://github.com/gestylinaga)

## Basic Syntax
```text
# Header1
## Header2
## Header3

*italics*
**bold**
***bold and italics***
~~strikethrough~~

[Link Title](https//linkurl.com)
![Image Title](path/to/image)
```

## Lists
```text
1. numbered item 1
    * bullet item 1
    * bullet item 2
        - dash item 1
        - dash item 2
2. numbered item 2
3. numbered item 3
    + plus item 1
        * bullet item
    + plus item 2
        * bullet item
```

## Tables
```text
| header | head | heading |
| ---    | ---  | ---     |
| text   | like | this    |

| header | head | heading |
| -      | -    | -       |
| or     | 1    | dash    |

| header     | head   | heading |
| ---:       | :---:  | :---    |
| 1          | 2      | 3       |
| x          | y      | z       |
| semicolons | change | spacing |
```

## Blockquotes
```text
> blockquotes
>
> like this
> 
>> nested
>> blockquotes too
>
> more text
```

## Checklists
```text
1. [ ] not done
2. [x] completed
3. [x] numbered lists

* [x] bullet lists
* [ ] like this

- [ ] dashes work 
- [x] too
```

## Horizontal Lines
```text
---
```
